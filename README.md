## Description

基于Nodejs + Nest + Typescript + MongoDB构建的一套Restfull风格的博客系统API。

## Development

```bash
git clone git@github.com:const-love-365-10000/blog_end.git
git ch develop
git pull
git ch -b [你自己的分支]
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# lint
$ npm run lint
```

## Feature

- [x] 基于JWT实现授权
- [x] 实现CORS跨域
- [x] Restfull风格API
- [x] 日志
- [x] 请求参数验证(守卫)
